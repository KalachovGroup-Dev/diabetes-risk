# Предсказатель риска развития сахарного диабета

Решение на базе машинного обучения для оценки риска развития диабета на основе клинических параметров.

Проект демонстрирует полный цикл разработки: от обработки несбалансированных медицинских данных до развертывания интерактивного веб-приложения для мгновенного анализа рисков.

## Предмет изучения 

Основная сложность при работе с такими данными — **сильный дисбаланс классов** (случаев диабета значительно меньше, чем здоровых пациентов). Ошибка пропуска больного пациента (False Negative) является более опасной, чем ложное срабатываение.

В данном проекте реализованы продвинутые стратегии обработки данных и настройки модели, чтобы минимизировать пропуски патологий и обеспечить высокую чувствительность предсказания.

## Краткое описание разработки

### 1. Предобработка данных
* Обработка категориальных признаков: Использование One-Hot Encoding для таких параметров, как пол и статус курения.
* Борьба с дисбалансом классов:
    * Применение метода **Downsampling** для выравнивания распределения.
    * Настройка гиперпараметра `scale_pos_weight` в модели XGBoost для увеличения веса редких (больных) случаев.
* Приведение числовых признаков к единому формату для корректной работы.

### 2. Разработка модели
* Алгоритм: **XGBoost**.
* Подбор гиперпараметров (`max_depth`, `learning_rate`) для предотвращения переобучения и захвата сложных нелинейных зависимостей между клиническими показателями.

### 3. Развертывание
* Разработка web-приложения с помощью **Streamlit**.
* Пользователь может вводить клинические параметры (возраст, давление, уровень холестерина и т.д.) и мгновенно получать оценку риска.
* Визуализация важности признаков в приложении, что позволяет понять, какие именно показатели повлияли на конкретный прогноз.
